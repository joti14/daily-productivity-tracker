# Daily Productivity Tracker

A simple, beginner-friendly React web app to help you stay productive every day. Track your tasks, build habits with streak tracking, and measure focus time; all in one clean dashboard.
Data is saved automatically using LocalStorage, so your progress persists even if you close or refresh the browser.

## Features

- Daily Tasks: Add, complete, and delete to-do items with a clean checklist.
- Habit Tracker: Create habits and build streaks by marking them complete each day.
- Focus Timer: A simple stopwatch to track focused work time (saved across sessions).
- Persistent Storage: Everything saves locally.
- Responsive & Clean UI: Easy to use on desktop or mobile.

## Technologies Used

- React (with Hooks: useState, useEffect)
- LocalStorage for data persistence
- Vanilla CSS for styling
